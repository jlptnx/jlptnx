## Responsive Considerations

All screens should be designed **mobile-first** with these breakpoints:

| Breakpoint | Width | Notes |
|------------|-------|-------|
| Mobile | < 640px | Primary design target |
| Tablet | 640-1024px | Side-by-side layouts where useful |
| Desktop | > 1024px | Max content width, centered |

Key mobile considerations:
- Bottom navigation for core actions
- Sheet/modal patterns for check-in
- Thumb-friendly tap targets (44px minimum)
- Single-column layouts by default
